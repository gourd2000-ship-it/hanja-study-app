# 한자 마로 (Hanja Maro) - 제품 설계 기록 (PDR)

## 1. 프로젝트 개요
'한자 마로'는 사용자가 한자 급수 시험(8급 ~ 6급)을 효율적으로 준비할 수 있도록 돕는 웹 기반 학습 애플리케이션입니다. 망각 곡선 이론에 기반한 복습 알고리즘과 테마별 한자 배치를 통해 학습 효과를 극대화합니다.

## 2. 주요 기능
### 2.1. 급수별 학습 커리큘럼
*   **제공 급수**: 8급, 7급A/B, 6급A/B/C (총 300자)
*   **테마별 배치 (6급)**: 가나다순이 아닌 의미 단위(동작, 자연, 사회 등)로 한자를 재배치하여 암기 효율 향상
*   **학습 스케줄**: 20일 완성 코스로 구성 (신규 학습 + 누적 복습)

### 2.2. 학습 및 퀴즈 시스템
*   **오늘의 한자**: 스케줄에 맞춰 매일 새로운 한자와 복습 한자 제공
*   **퀴즈**: 4지선다형 퀴즈를 통해 학습 내용 점검
*   **오답 관리**: 틀린 한자를 자동으로 '약점 한자'로 등록하여 집중 학습 지원

### 2.3. 진행 상황 관리
*   **로컬 데이터 저장**: `localStorage`를 사용하여 별도의 로그인 없이 브라우저에 학습 진도 저장
*   **마이그레이션**: 앱 업데이트 시 기존 데이터를 보존하며 새로운 급수 데이터를 자동으로 추가

## 3. 기술 스택
*   **Frontend**: React, TypeScript, Vite
*   **Styling**: Vanilla CSS / Tailwind CSS
*   **State Management**: Custom React Hook (`useStudy`)
*   **Testing**: Vitest, React Testing Library
*   **Deployment**: Vercel

## 4. 아키텍처
*   **Data Layer (`src/data/hanja.ts`)**: 전체 한자 데이터 및 인터페이스 정의
*   **Logic Layer (`src/hooks/useStudy.ts`)**: 학습 스케줄 계산, 진도 저장, 퀴즈 로직 등 핵심 비즈니스 로직 담당
*   **UI Layer (`src/App.tsx`)**: 사용자 인터페이스 렌더링 및 인터랙션 처리

## 5. 향후 과제 (Future Roadmap)
### 5.1. 기능 확장
*   **고급 급수 추가**: 5급 이상의 한자 데이터 및 커리큘럼 확장
*   **멀티미디어 학습**: 한자 획순 애니메이션 및 원어민 발음 기능 추가
*   **다양한 퀴즈 모드**: 필기 인식 퀴즈, 매칭 게임 등 흥미 유발 요소 추가

### 5.2. 기술 개선
*   **클라우드 동기화**: Firebase 또는 Supabase를 연동하여 기기간 학습 데이터 동기화
*   **PWA 지원**: 오프라인에서도 학습이 가능하도록 Progressive Web App 도입
*   **통계 대시보드**: 학습 시간, 정답률 추이 등을 시각화하여 제공

### 5.3. 사용자 경험 (UX)
*   **게이미피케이션**: 연속 학습 일수(Streak), 배지 시스템 도입으로 동기 부여
*   **다크 모드 지원**: 사용자 선호에 따른 테마 선택 기능
